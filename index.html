VOICE GENDER PREDICTOR

BUILDING A GENDER RECOGNITION VOICE APP

CHECK IT OUT

Description

This project was all about building a Gender Recognition by Voice app, using a Random Forest classifier algorithm with sklearn in Python. I used a dataset of voice features to train my model, so that it can predict the gender of a voice based on its characteristics. And I'm really happy to say that my model has an impressive 99% accuracy rate! To make things even more user-friendly, I also created a Streamlit app that lets users input their own voice data through sliders for different features, and then see the predicted gender.
